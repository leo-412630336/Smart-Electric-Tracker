##  專案簡介
電費紀錄系統是一個簡易的網頁應用，使用者可紀錄每月的用電量與電費。  
系統會自動依據 **台電的累進電價公式** 計算出實際費用與平均電價，  
並估算相對應的 **碳排放量**，幫助使用者了解能源使用對環境的影響。  



##  使用技術
- **HTML**：負責頁面結構與內容呈現  
- **CSS**：設計版面與深色／淺色模式樣式  
- **Bootstrap**：強化排版與元件設計  
- **JavaScript**：負責電價計算、資料儲存與互動邏輯  
- **Constraint Validation API**：表單輸入驗證機制  

---

##  功能特色
1. 使用者可輸入每月用電量，系統自動依台電電價公式計算出電費。  
2. 支援『電價分段累進制計算』，更貼近實際電費。  
3. 可記錄多個月份的用電資料並自動整理成歷史紀錄。  
4. 提供總電費、平均電價、總碳排放等統計資訊。  
5. 具備 『深色模式切換』 與 『節能小提示』 功能。
